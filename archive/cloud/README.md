# Archived Cloud Deployment Assets

This directory contains historical SaaS/cloud deployment assets retained for reference.

Status as of 2026-03-28:
- ELMS product deployment is local-only.
- Cloud onboarding and cloud runtime are non-operational.
- Files here are intentionally excluded from active developer and release workflows.

Contents:
- `apps/web/`: former Docker/Nginx web deployment wrapper.
- `scripts/deploy-cloud.sh`: former cloud deploy helper.
- `scripts/backup-postgres.sh`, `scripts/restore-postgres.sh`: former cloud compose DB scripts.
